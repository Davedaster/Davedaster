# Customer tracking progress milestone

This milestone improves the public customer tracking page so it feels more like a useful delivery tracker.

## Added

- Route progress percentage
- Completed drop count
- Remaining drop count
- Customer drop number
- Stops before the customer
- Clear customer status message
- Safer fallback messages for planned, active, completed and inactive routes

## Customer experience

Customers can now see where their order sits in the route without exposing the full driver route or other customers' details.

The page still keeps live driver visibility limited to the next active drop only, which protects privacy while giving the current customer a useful update.

- Add a real map provider when the map milestone is ready
- Add SMS and email links to this tracking page
- Add a refreshed ETA when automatic routing is added
